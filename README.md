# Library App

Demo App for searching various books, see their details, read what reviews they received from the New York Times and also save them as your favourties. 
App is based on New York Times API and Google Books API 

### Learning objectives of Project
- Use fetch API calls 
- use React Router to handle the various pages 
- better component architecture
